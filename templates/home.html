Student Performance Indicator

About This Project

This Machine Learning project predicts a student's math score based on several factors such as gender, ethnicity, parental education level, lunch type, test preparation course, reading score and writing score.

The goal of this project is to understand how different factors affect student academic performance.

Life Cycle of Machine Learning Project

  • Understanding the Problem Statement
  • Data Collection
  • Data Checks to Perform
  • Exploratory Data Analysis
  • Data Pre-Processing
  • Model Training
  • Choose Best Model

Problem Statement

This project analyzes how the student's performance (test scores) is affected by other variables such as:

  • Gender
  • Race or Ethnicity
  • Parental Level of Education
  • Lunch Type
  • Test Preparation Course

Dataset Information

  • Dataset contains 1000 rows and 8 columns
  • Student performance dataset from Kaggle
